'local'
Если 'local' в верхнем меню подсвечивается зеленым, все вычисления происходят с помощью вашего браузера или, если по другому говорить, с помощью вашего компьютера.

'server'
Если 'server' в верхнем меню подсвечивается зеленым, все вычисления происходят на нашем сервере. Стоит использовать если вы зашли на сайт через такойже калькулятор :P

'pow'
Возводит число в степень. Команда выглядит так: 'pow(a,b)'. Где 'a' - значение которое возводится в степень, 'b' - значение в которое возводится 'a'. Если ввести эту команду неверно, произойдет ошибка.

'sqrt'
Получает корень числа. Команда выглядит так: 'sqrt(a)'. Гду 'a' - значение от которого ищется корень. Если ввести эту команду, произойдет ошибка.

'C'
Стирает последний введенный символ.

'CE'
Стирает все введенные символы.

'copy'
Копирует все данные с дисплея используя буфер обмена. Внимание! Нажимая эту кнопку ваш буфер обмена будет изменен и данные которые вы копировали раньше пропадут.

'paste'
Вставляет в дисплей данные из буфера обмена. Внимание! Чтобы использовать эту кнопку, нужно разрешить сайту чтение буфера обмена. 

','
Запятая используется только в команде 'pow'. Используя запятую вне указаной команды будет вызвана ошибка.

'*'
Данный символ предназначен для умножения чисел. Выглядит как: 'a*b'.

'/'
Данный символ предназначен для деления чисел. Выглядит как: 'a/b'.

'+'
Данный символ предназначен для добавления чисел. Выглядит как: 'a+b'.

'-'
Данный символ предназначен для отнимания чисел. Выглядит как: 'a-b'.
